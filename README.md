# taksh.sh // Terminal Portfolio

An interactive, Unix-style terminal portfolio built entirely from scratch with vanilla web technologies to demonstrate core programming logic, state management, and custom DOM engineering. 

---

## 🛠️ The Vision

Instead of using heavy UI frameworks or copy-pasting standard templates, this project serves as an "AI-proof" demonstration of foundational web development. Visitors are greeted by a custom 3D entrance animation before dropping into a hybrid shell interface capable of navigating pages, parsing arguments, and executing commands.

### Key Features
*   **3D Kinematics Intro:** A pure CSS/JS entry animation where characters rotate on a $X$-axis 3D plane while sliding into alignment.
*   **Custom Command Parser:** Built-in shell interpreter to handle system arguments, page routing, and history buffers.
*   **Hybrid Architecture:** Seamlessly switches from a pure CLI interface into content pages (like achievements or projects) while maintaining a persistent terminal dock at the bottom.

---

## 🏗️ Technical Stack

*   **Structure:** HTML5 (Semantic Layout)
*   **Styling:** CSS3 (3D Transforms, Custom Properties/Themes, Flexbox Arena)
*   **Logic:** Vanilla JavaScript (ES6+, DOM Manipulation, Array Mapping, Event Receivers)
